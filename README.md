# API易 AI对话

本地运行的HTML网页，调用API易的OpenAI兼容接口进行AI对话。

## 使用方法

1. 点击 `index.html` 文件
2. 点击右上角 Raw 按钮下载
3. 用浏览器打开下载的HTML文件即可使用

## 功能

- 支持 GPT / Claude / Grok / Gemini / DeepSeek 全系列模型
- 流式/非流式输出
- 多轮对话
- 思考过程折叠显示
- Markdown渲染
- 适配安卓手机和平板
- 零依赖，纯本地运行

## 在线体验

打开 [GitHub Pages](https://owjk123.github.io/apiyi-chat/) 可直接在浏览器中使用。
